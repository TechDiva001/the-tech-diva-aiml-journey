# 🏆 Ghana Galamsey Impact Tracker — SQL Capstone

## Overview
Relational database and SQL analytics project exploring the environmental and public health impact of illegal mining (galamsey) in Ghana.

## Dataset
Research-backed water quality, mining activity, water supply, and public health data for all 16 regions of Ghana (2020–2026).

## What I Did
- Designed a normalized SQLite relational database
- Created and linked 6 relational tables with foreign keys
- Wrote SQL queries using joins, aggregations, subqueries, and views
- Analyzed water quality, mining activity, and health impacts
- Built interactive visualizations to communicate key insights

## Tools Used
SQLite, SQL, Python, Pandas, Plotly, Matplotlib, Seaborn, Jupyter Notebook
