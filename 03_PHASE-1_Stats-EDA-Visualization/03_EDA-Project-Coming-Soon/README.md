# 🧹 CSV Auto-Cleaner — EDA Capstone

**Built by Benedicta Bondzie | The Tech Diva**

---

## Overview
An interactive, step-by-step CSV data cleaning tool built as a capstone project after completing the Exploratory Data Analysis phase of my self-taught AI/ML journey. 

Upload any messy CSV file, clean it through guided prompts, and download it ready for analysis. No code to write.

---

## What I Did
- Built an end-to-end interactive data cleaning tool that runs fully in Google Colab
- Handled missing values in both numeric and categorical columns with 10+ cleaning options
- Built duplicate detection and removal with multiple keep modes
- Detected outliers using IQR Method and Z-Score Method with box plot visualizations
- Built GroupBy analysis for both numeric and categorical columns with charts
- Built a correlation matrix using NumPy and Pandas with a heatmap visualization
- Written full input validation and error handling for every user prompt
